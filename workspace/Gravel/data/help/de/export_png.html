Abbildung 1: Einstellungen des PNG-Exports nach Wahl einer Datei

Als Export in eine Pixelgrafik bietet Gravel den PNG-Export. Dabei wird eine Grafik im Format Portable Network Graphics erstellt. Dies wird durch Java-interne Bibliotheken ermöglicht. Das Format ist verlustfrei.
Nach Wahl einer Datei im Export-Dialog erscheint ein weiteres Fenster (Abbildung 1). Hier lässt sich entweder eine Breite oder eine Höhe des Bildes angeben. Die Angabe ist in Pixel. Der nicht direkt angegebene Wert wird dann durch das Seitenverhältnis des Graphen bestimmt. Für das Seitenverhältnis werden die ungenutzten Ränder der Darstellung entfernt. Es wird lediglich die Fläche verwendet, die der Graph auch wirklich benötigt. Bei der Berechnung werden neben Kanten und Knoten auch Knotennamen und Kontrollpunkte berücksichtigt.
Ist die Größe festgelegt, wird der Graph als Bild exportiert. Dabei werden alle Farben bedacht. Die Einstellungen der aktuellen Anzeige werden berücksichtig; Raster und Kontrollpunkte werden ebenfalls exportiert, so diese aktiviert sind. Außerdem ist wie in der Anzeige eine Kantenglättung aktiv, das heißt, dass die Ränder der gezeichneten Objekte geglättet werden. Der Hintergrund wird auf transparent gesetzt.

Einsatzbereich dieses Exportes sind alle Weiterverwendungen außerhalb von Gravel und LaTeX
