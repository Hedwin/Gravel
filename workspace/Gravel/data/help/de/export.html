Export

Der Export stellt Methoden für Dateiformate zur Verfügung, um den erstellten Graphen außerhalb von Gravel weiter zu verwenden. Dabei wird zunächst eine Datei über den entsprechenden Dialog abgefragt. Anschließend werden die dem Export-Format entsprechenden Werte abgefragt.

Bisher existieren die folgenden Exporte:
  • Portable Network Graphics
  • LaTeX-Picture-Umgebung
